# 🎮 Jogo da Velha em Java Swing

Este projeto consiste no desenvolvimento de um **Jogo da Velha com interface gráfica**, utilizando a biblioteca **Java Swing**.
O sistema foi desenvolvido com o objetivo de praticar conceitos fundamentais de **Programação Orientada a Objetos**, **eventos gráficos**, **lógica de programação** e **versionamento com Git/GitHub**.

---

## 📌 Sobre o projeto

O Jogo da Velha é um jogo clássico para dois jogadores onde cada participante alterna sua jogada entre os símbolos:

* ❌ Jogador X
* ⭕ Jogador O

O objetivo é completar **uma linha, coluna ou diagonal** com o mesmo símbolo antes do adversário.

O sistema foi desenvolvido com interface gráfica desktop, permitindo interação através de botões que representam o tabuleiro.

---

## 🚀 Funcionalidades implementadas

* Alternância automática entre os jogadores **X e O**
* Bloqueio de jogadas em posições já preenchidas
* Verificação automática de vitória nas:

  * Linhas
  * Colunas
  * Diagonais
* Detecção de empate
* Exibição de mensagens informando o vencedor
* Botão para reiniciar o jogo
* Interface gráfica amigável utilizando Java Swing
* Controle de estado da partida
